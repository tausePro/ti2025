# Assets de Marca - Talento Inmobiliario

Este directorio contiene los assets de marca para el membrete de los informes PDF.

## Archivos requeridos

### Logos
- **logo-header.png** - Logo principal para el encabezado (recomendado: 300x100px aprox)
- **logo-footer.png** - Logo circular para el pie de página (recomendado: 80x80px)

### Opcionales (mejor calidad)
- **logo-header.svg** - Versión SVG del logo principal
- **logo-footer.svg** - Versión SVG del logo circular

## Configuración

El archivo `brand-config.json` contiene:
- Colores de marca
- Datos de contacto
- Rutas a los assets

## Uso

Los informes PDF generados usarán automáticamente estos assets para el membrete.
